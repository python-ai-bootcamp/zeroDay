zeroDayBootcamp is an israelli initiative to teach and encourage children across the country to learn and engage in software development

In the boot camp you will learn the following:

  • Basics of computer Programing
  • Python Programing Language
  • Various transferable software industries skills (IDE usage, Bash scripting etc...)
  • Deep Knowledge about AI Programing and manipulation for various tasks

All will be tought in a fun interactive way:

  • Weekly Online Technical Viewing Material
  • Weekly Guided Zoom Sessions
  • Weekly Task testing knowledge of last weeks material
  • Each student can continue to next session only once he completed the previous task successfully

The survivers of the entire bootcamp will be invited to join a recreational R&D group:

  • Developing open source code in a guided scenario
  • Reasearch and development of novel AI algorithms
  • Writing scientific papers
